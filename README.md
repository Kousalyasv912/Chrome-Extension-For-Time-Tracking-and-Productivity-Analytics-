# Overview  
FocusFlow Dashboard is a productivity tracking application designed to help individuals and teams understand how their time is spent online. Built with React, Express, and MongoDB, it provides a clean, professional interface that categorizes minutes into productive and unproductive. The dashboard uses a dark theme with violet branding to ensure readability and a polished look suitable for client presentations. By combining tabular data with interactive charts, FocusFlow makes productivity insights both clear and actionable.

# Installation
1.Clone the repository.

2.Navigate to both frontend and backend folders and run npm install.

3.Start the backend with node server.js.

4.Start the frontend with npm run dev.

5.Open http://localhost:5173 in your browser.

# Usage  
Once launched, the dashboard displays a table of sites visited along with productivity metrics. Users can refresh data at any time and view insights through Pie or Bar charts. The Pie Chart provides a quick overview of productive vs. unproductive time, while the Bar Chart breaks down site‚Äëspecific usage. Buttons below the chart allow users to switch between views without scrolling, keeping the interface concise and professional.
